# Packer + Obfuscator

A low-level executable packer written in C++
## Features

Combine executables into a single file
Generate as EXE or DLL

### Prerequisites

- Visual Studio 2019+ (Windows)
- CMake 3.20+
- Qt 6.x
- Windows SDK

### Build Steps

```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release
```
